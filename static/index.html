Análisis Clínico Inteligente

Un endocrinólogo virtual analiza tus registros y te da recomendaciones personalizadas. Necesitas al menos 5 registros y una API key de Anthropic.

⚠️ El análisis es generado por IA y tiene fines informativos. NO reemplaza la consulta con tu endocrinólogo. Comparte estos datos con tu equipo médico para decisiones clínicas.
⚡
Documentación de la API

Comparte esta API con otros diabéticos. Todos los endpoints están listos para integrar.

POST/api/v1/bolus

Calcula dosis de bolus de insulina

GET/api/v1/alimentos

Lista todos los alimentos

GET/api/v1/alimentos/buscar?q=arroz

Busca alimento por nombre

POST/api/v1/estimar-carbohidratos

Estima CHO por lista de alimentos

POST/api/v1/analizar-imagen

Analiza foto con IA

Ejemplo cURL

curl -X POST http://localhost:8000/api/v1/bolus \
  -H "Content-Type: application/json" \
  -d '{
    "perfil": {
      "nombre": "Carlos",
      "ratio_insulina_carbohidratos": 10,
      "factor_sensibilidad": 40,
      "glicemia_objetivo": 110,
      "tipo_terapia": "MDI"
    },
    "glicemia_actual": 180,
    "carbohidratos": 60
  }'
